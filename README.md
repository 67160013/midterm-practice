# Simple Calculator Web App

Calculator Web App สำหรับฝึกสอบกลางภาค  
พัฒนาด้วย HTML, CSS และ JavaScript

## Features

- บวก ลบ คูณ หาร
- ปุ่ม Clear (AC)
- เก็บประวัติการคำนวณ (ล่าสุดอยู่บน)
- คำนวณอัตโนมัติเมื่อกด operator
- ป้องกัน Division by Zero
- รองรับปุ่ม Enter (Keyboard)
